Key features

Thought Reframing

Card-based interface with curated quotes to gently shift perspective.

Category Filters

Choose categories that matter to you. First launch guides you to set them up.

Bookmarks

Favorite quotes and quickly revisit them. Manage them from a dedicated view.

Offline Support

All data is stored locally on your device using Core Data. No account required.

Home Screen Widget

See rotating quotes on your Home Screen. Supports small and medium sizes.

Respectful Analytics

Anonymous usage metrics help improve the app. No personal data is collected.
