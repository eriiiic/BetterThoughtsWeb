A 10‑Minute Morning Mindset Routine (CBT Edition)

January 22, 2025 · 4 min read

1. Breath Reset (1 min)

Inhale 4, exhale 6. Soften shoulders and jaw.

2. Values Check (2 min)

Write one value for today (e.g., kindness, focus) and a micro‑action that embodies it.

3. Thought Reframe (3 min)

Capture one worry and write a balanced alternative thought.

4. Behavioral Activation (3 min)

Choose one 5‑minute starter task. Start small; build momentum.

5. Gratitude Anchor (1 min)

Note one thing you appreciate right now to widen attention.
