CBT for Procrastination: Behavioral Activation That Works

January 24, 2025 · 5 min read

Make It Startable

Define the first visible action (open doc, title page). If it takes over 2 minutes, it’s not first.

Time‑Box with Mercy

Try a 5–10 minute timer. End regardless of progress; success is showing up.

Design the Path

Put tools in reach, remove friction, and create a ritual (tea, playlist, do‑not‑disturb).

Talk to Yourself Kindly

Swap “I should be further” with “I’m building momentum; small steps count.”
